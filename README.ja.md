# 発言アシスタント

[English](README.md) ｜ [中文简体](README.zh-CN.md) ｜[中文正體](README.zh-TW.md) ｜日本語｜[한국어](README.ko.md)

## 概要

発言アシスタントは、「スーパー入力メソッド」のようなスマート発言補助ツールです。
過去の発言内容をもとに連想・補完を行い、よく使う返信をより速く、自然に入力できます。

使い方を変える必要はなく、すべてのウェブサイト、すべての言語、すべての入力方式に対応しています。インストール後すぐに利用可能です。

すべてのデータはローカル端末内にのみ保存され、サーバーを経由せず、アップロードも行いません。プライバシー漏洩のリスクを完全に回避します。

[プライバシーポリシー](Privacy-Policy.md)

## 主な機能

- 現在のサイトの文脈、キーワード、ページタイトルに基づく発言履歴・参照内容の自動連想
- 絵文字・よく使う単語のクイックサジェスト
- 発言履歴の自動記録と基本的な管理機能
- カスタム参照コンテンツのインポート対応
- 多言語対応

## ショートカット操作

- **Tab / Shift + Tab**：連想入力の有効化
- **スペースキー**：現在の候補を確定
- **Esc**：連想表示を閉じる
- **Backspace**：特定条件下で連想状態を終了

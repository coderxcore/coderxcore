# 발언 어시스턴트

[English](README.md) ｜ [中文简体](README.zh-CN.md) ｜[中文正體](README.zh-TW.md) ｜[日本語](README.ja.md)｜한국어

## 소개

발언 어시스턴트는 ‘슈퍼 입력기’와 유사한 지능형 발언 보조 도구입니다.
과거 발언 내용을 기반으로 연상 및 자동 완성을 제공하여, 자주 사용하는 응답을 더 빠르고 자연스럽게 입력할 수 있습니다.

사용 습관을 바꿀 필요 없이 모든 웹사이트, 모든 언어, 모든 입력 방식과 호환되며 설치 후 바로 사용할 수 있습니다.

모든 데이터는 로컬 기기에만 저장되며 서버를 거치거나 업로드되지 않습니다. 개인정보 유출 위험을 완전히 차단합니다.

[개인정보 정책](Privacy-Policy.md)

## 주요 기능

- 현재 웹사이트의 상황, 키워드 또는 페이지 제목을 기반으로 한 발언 기록 및 참고 내용 자동 연상
- 이모지 및 자주 사용하는 단어의 빠른 추천
- 발언 기록 자동 저장 및 기본 관리 기능
- 사용자 정의 참고 콘텐츠 가져오기 지원
- 다국어 지원

## 단축키

- **Tab / Shift + Tab**：연상 기능 활성화
- **스페이스바**：현재 후보 확정
- **Esc**：연상 표시 닫기
- **Backspace**：특정 조건에서 연상 상태 종료

## 향후 업데이트 계획

- AI 기반 스마트 발언 기능 도입
- AI가 여러 개의 참고 응답 자동 생성
- Supabase 지원: 다중 기기 동기화, 데이터 백업
- 발언 기록 관리 기능 강화
- 연상 알고리즘 개선으로 정확도 향상 및 방해 요소 감소
- 입력창 유형별 다른 제안 제공 및 개별 비활성화 옵션
- 더 세밀한 사용 시나리오 관리

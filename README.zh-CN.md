# 发言助手

[English](README.md) ｜ 中文简体 ｜[中文正體](README.zh-TW.md) ｜[日本語](README.ja.md)｜[한국어](README.ko.md)

## 简介

发言助手是一款类似“超级输入法”的智能发言辅助工具。
它会基于你以往的发言内容进行联想与补全，让常用回复更快、更自然。

无需改变你的使用习惯，兼容所有网站、所有语言、所有输入法，即装即用。

所有数据仅保存在本地设备，不经过服务器、不上传，完全避免隐私泄漏风险。

[隐私政策](Privacy-Policy.md)

## 主要功能

- 根据当前网站场景、关键词或页面标题，智能联想历史发言与导入的参考内容
- 表情与常用词语快捷提示
- 自动记录发言历史，并支持基础管理
- 支持导入自定义参考内容
- 多语言支持

## 快捷操作

- **Tab / Shift + Tab**：启用快捷键
- **空格**：确认当前候选
- **Esc**：关闭联想提示
- **Backspace**：在特定情况下退出联想状态

# 發言助手

[English](README.md) ｜ [中文简体](README.zh-CN.md) ｜中文正體 ｜[日本語](README.ja.md)｜[한국어](README.ko.md)

## 簡介

發言助手是一款類似「超級輸入法」的智慧發言輔助工具。
它會根據你過往的發言內容進行聯想與補全，讓常用回覆更快速、更自然。

無需改變原本的使用習慣，相容所有網站、所有語言與所有輸入法，安裝後即可使用。

所有資料僅儲存在本機裝置，不經過任何伺服器、不上傳，完整避免隱私外洩風險。

[隱私政策](Privacy-Policy.md)

## 主要功能

- 根據目前網站情境、關鍵字或頁面標題，智慧聯想歷史發言與匯入的參考內容
- 表情符號與常用詞彙快速提示
- 自動記錄發言歷史，並提供基礎管理功能
- 支援匯入自訂參考內容
- 多語言支援

## 快捷操作

- **Tab / Shift + Tab**：啟用聯想快捷鍵
- **空白鍵**：確認目前候選內容
- **Esc**：關閉聯想提示
- **Backspace**：在特定情況下退出聯想狀態
